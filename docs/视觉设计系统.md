# 项目视觉设计系统

本文档定义“摄影小助手”的视觉语言、设计 token、组件规范与页面迁移规则。当前系统以曝光计算器的暗场工具界面为主基准，后续景深、色温等页面应逐步向同一套视觉系统收敛。

## 1. 设计定位

产品气质：现场拍摄工具，而不是内容型网站或营销页。

关键词：

- 暗场：适合夜景、长曝光、户外拍摄场景。
- 精确：数值是第一视觉层级，界面辅助计算而不抢戏。
- 克制：少卡片、少装饰、少说明文字。
- 可触控：高频操作目标大，左右切换可盲操作。
- 低干扰：黑底、白字、灰色辅助信息，黄色只用于主动作。

## 2. 设计原则

1. 数值优先：结果时间、当前档位、当前补偿值必须比标题和说明更醒目。
2. 一屏完成：移动端优先让主流程在首屏内完成；无法完全容纳时也不得出现横向滚动。
3. 三段式选择：连续档位优先使用“上一项 / 当前项 / 下一项”的滚轮结构，而不是普通下拉框。
4. 主操作唯一：每个核心工具页只保留一个强主操作，曝光页为黄色圆形“开始”。
5. 文案少而准：页面上不放功能介绍、使用教程或键盘快捷键说明。
6. 工具和内容分离：高级信息放设置层、抽屉或次级区域，不挤占主计算路径。

## 3. 颜色 Token

| Token | 值 | 用途 |
| --- | --- | --- |
| `--color-canvas` | `#000000` | 暗色工具页背景 |
| `--color-surface` | `#101010` | 设置浮层、深色面板 |
| `--color-surface-subtle` | `#f9fafb` | 浅色页面中的弱面板 |
| `--color-text-primary` | `#ffffff` | 暗色页主文字、核心数值 |
| `--color-text-strong` | `#111827` | 浅色页主文字 |
| `--color-text-secondary` | `#f3f4f6` | 暗色页次级文字 |
| `--color-text-muted` | `#9ca3af` | 辅助说明、弱信息 |
| `--color-text-disabled` | `#414141` | 滚轮相邻项 |
| `--color-text-disabled-subtle` | `#343434` | 滚轮相邻项副标题、浮层边线 |
| `--color-divider-strong` | `#676767` | 暗色页主分割线 |
| `--color-divider-subtle` | `#323232` | 滚轮定位三角 |
| `--color-accent-primary` | `#f6b700` | 主按钮、关键触发动作 |
| `--color-accent-primary-pressed` | `#d99000` | 主按钮按下、计时运行态 |
| `--color-danger` | `#ef4444` | 危险操作、错误 |
| `--color-success` | `#10b981` | 成功、正向容差 |

使用规则：

- 黄色只能用于主操作和必要的激活反馈，不用于普通装饰。
- 暗色页不要使用大面积蓝、紫、棕色渐变。
- 分割线可见但不能抢眼，优先使用 `--color-divider-strong`。
- 相邻档位必须明显弱于当前档位，使用 `--color-text-disabled`。

## 4. 字体与数字

全项目默认使用系统字体：`system-ui, sans-serif`。

| 样式 | 字号 / 行高 / 字重 | 用途 |
| --- | --- | --- |
| App 标题 | `34px / 40px / 800` | 暗色工具页标题 |
| 小屏 App 标题 | `28px / 34px / 800` | 宽度小于 380px |
| 滚轮当前值 | `42px / 46px / 850` | 快门、补偿当前值 |
| 滚轮 ND 当前值 | `34px`，小屏 `30px` | 适配 `ND512000` 等长标签 |
| 滚轮相邻值 | `26px / 32px / 800` | 左右相邻档位 |
| 字段标题 | `24px / 30px / 500` | 滤镜档位、基础快门、曝光补偿 |
| 结果说明 | `28px / 36px / 500` | “装入滤镜后的快门时间” |
| 结果数值 | `58px / 64px / 850` | 主结果时间 |
| 小屏结果数值 | `46px / 52px / 850` | 宽度小于 380px |
| 主按钮文字 | `34px / 40px / 500` | 黄色圆形按钮 |

数字规则：

- 时间、EV、档位结果使用 `font-variant-numeric: tabular-nums`。
- 大结果超过 60 秒时显示 `h:mm:ss`，例如 `364:05:20`。
- 基础快门滚轮中，整秒项显示为 `2.5`、`1`、`30`；分数秒保留 `1/250s`。
- 不使用随视口宽度缩放的字体，改用断点和固定字号。

## 5. 间距与布局

全局：

- 主内容最大宽度：普通页 `420px`，曝光暗色页 `480px`。
- 普通页边距：`--page-gutter: clamp(8px, 3vw, 24px)`。
- 暗色曝光页边距：常规 `24px`，小屏 `18px`。
- 小屏断点：`max-width: 380px`。

曝光页结构：

- 顶部栏：三列 `56px / 1fr / 56px`，小屏 `48px / 1fr / 48px`。
- 滚轮：三列 `0.8fr / 170px+ / 0.8fr`，小屏中列至少 `174px`。
- 滚轮常规高度：`126px`，小屏 `94px`。
- 主按钮常规尺寸：`144px`，小屏 `118px`。
- 结果区居中，不嵌套卡片。

## 6. 形状、线条与阴影

| Token | 值 | 用途 |
| --- | --- | --- |
| `--radius-control` | `8px` | 普通控制、卡片、浮层 |
| `--radius-round` | `999px` | 圆形和胶囊形控件 |
| `--shadow-accent-action` | `rgba(246, 183, 0, 0.28) 0 16px 36px` | 主按钮光晕 |
| `--shadow-popover` | `rgba(0, 0, 0, 0.38) 0 16px 36px` | 深色浮层 |

规则：

- 工具页不要把页面 section 做成浮动大卡片。
- 单个重复项、浮层、结果组可以使用卡片。
- 暗色页以线条和留白分区，不靠边框盒子堆叠。
- 圆形按钮只用于开始、拍摄、计时这类核心动作。

## 7. 组件规范

### 顶部工具栏

结构：左操作、居中标题、右操作。

- 左侧通常为返回。
- 右侧通常为设置。
- 图标按钮最小触控尺寸 `44px`，曝光页当前为 `52px`。
- 图标必须有 `aria-label`。

### 滚轮选择器

结构：

- 顶部字段标题。
- 上分割线和向下三角。
- 三列选项：上一档、当前档、下一档。
- 下分割线和向上三角。

状态：

- 当前值白色、最大字号。
- 相邻值深灰，可点击。
- 副标题用于 stops、单位或辅助说明。
- 长标签可以压缩字号，但不应省略当前值。

适用场景：

- ND 档位。
- 快门档位。
- 曝光补偿。
- 后续可扩展到光圈、ISO、焦距的离散档位。

### 结果区

曝光页结果区由标题、主时间、主按钮组成。

- 标题居中。
- 主结果使用白色大字。
- 长曝光状态显示黄色圆形按钮。
- 运行中按钮文案为“暂停”，颜色变为 `--color-accent-primary-pressed`。
- 运行后出现“重置”次级按钮。

### 设置浮层

- 从右上角触发。
- 背景使用 `--color-surface`。
- 边线使用 `--color-text-disabled-subtle`。
- 只承载高级、低频、诊断型信息。
- 不阻断主流程。

### 普通浅色页组件

景深、色温等页面可以暂时保留浅色工具风格，但应逐步统一：

- 控制区使用清晰分组。
- 结果数值比说明更醒目。
- 卡片圆角不超过 `8px`。
- 不使用营销式 hero、装饰图形或大面积插画。

## 8. 交互状态

| 状态 | 视觉 |
| --- | --- |
| 默认 | 主文字白色或深色，背景克制 |
| 按下 | 主色变深，或文字变主色 |
| 运行中 | 主按钮使用 `--color-accent-primary-pressed` |
| 禁用/相邻 | 深灰文字，不出现强边框 |
| 成功 | `--color-success` |
| 危险 | `--color-danger` |

交互规则：

- 高频按钮必须能一眼识别。
- 设置、返回等次级操作保持透明背景。
- 不在主页面放冗长帮助说明；必要说明放文档或设置层。

## 9. 可访问性与移动端规则

- 触控目标最小 `44px`。
- 暗色页主文字对比度必须接近白底黑字级别。
- 所有图标按钮必须提供 `aria-label`。
- 结果变化区域使用 `aria-live` 或稳定的测试定位。
- 不允许横向滚动。
- 小屏需要单独压缩高度和字号，不能靠浏览器自动缩放。
- 时间和数值不得被按钮、分割线或系统安全区遮挡。

- `src/presentation/global.css` 的 `:root`

- `src/presentation/theme.ts`

- `src/presentation/screens/ExposureScreen.tsx`

2. 将景深页的滑块结果区改为更清晰的数值层级。
3. 将通用按钮、设置浮层、结果数值样式抽为复用组件。
4. 将剩余硬编码颜色逐步替换为设计 token。
